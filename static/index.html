Supply Chain Inventory Simulation

The Supply Chain Inventory Management system is a full-stack, real-time application that models the dynamic lifecycle of goods transitioning through a warehouse. It features an automated backend simulation engine built with Python and FastAPI that processes live shipping depletion logic and controls autonomous supplier reorder triggers.

Dual-Source Architecture

  • Redis (In-Memory Micro-Cache): High-velocity source of truth for active stock telemetry, dynamic threshold attributes, and priority backorder queues.
  • Apache Cassandra: Immutable storage ledger for deep analytics, recording inventory snapshots, supplier orders, and telemetry logs.

Frontend Experience

The modern single-page UI layer uses HTML, JavaScript, and CSS to let users modify items, run live simulation scenarios, and inspect metrics across a clean dashboard and log summary views.

Backend Logic Overview

  • REST Endpoints: Handles API ingestion for items, database access, and simulation state.
  • Simulation Engine: Runs asynchronous loops that update stock levels, triggers alerts, and queues reorder events.
  • Order & Inventory Tracking: Maintains ledger fidelity while dynamically reflecting stock changes in the cache and persistent store.
